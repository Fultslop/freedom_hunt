# Challenge Forms Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add data-driven YAML-configured forms to challenge cards that proxy submissions through the Cloudflare Worker to a Google Sheet via Apps Script.

**Architecture:** A new `ChallengeForm` component reads field definitions from `location.challenge.form` in YAML, validates input client-side, and POSTs JSON to `/form-submit`. The existing Cloudflare Worker gains that route and forwards the payload to a Google Apps Script URL stored as a Worker secret (`FORM_SCRIPT_URL`). The Apps Script appends one row per submission to a Google Sheet.

**Tech Stack:** React 19 (JSX, inline styles), Vitest + @testing-library/react, Cloudflare Workers, Google Apps Script
